Exploring hover effect posibilities

::before / ::after, :hover and CSS transitions

I recently started creating a collection of components and effects. Having a backlog of ideas at hand will come in handy at one point or another. But above all, in creating this collection, I got a chance to explore the possibilities of the ::before and ::after pseudo-classes, in combination with :hover state and transitions.

Each of the divs below has a different hover effect. Checkout the source code or fork the Pen to use any of them in your own projects.

See the Pen Hover effect divs (transitions) by jvsandick (@Yeaust) on CodePen.

Notes

This is collection is solely made as a reference guide. Sizes, colors, transition directions and speeds are just for demonstration purposes. Change them as you seem fit.
